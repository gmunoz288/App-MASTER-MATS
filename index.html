Instrucciones de uso

        

        

          
PASO 1 - Cargar ficheros: Selecciona el cliente, el Workpackage-Summary y la Base de datos antes de ejecutar.

          
PASO 2 - Ejecutar MASTER: Genera la tabla MASTER con las tareas del Workpackage cruzadas con la Base de datos. Ejecutar siempre primero.

          
PASO 3 - Ejecutar MAT: Genera la tabla MAT con los materiales y utillaje (MAT=SI) de cada tarea. Requiere MASTER ejecutado previamente.

          
PASO 4 - Revisar resultados: Las filas en ROJO requieren control manual. Edita las celdas que quieras actualizar en la Base de datos; quedan marcadas en VERDE.

          
PASO 5 - Actualizar Base de datos: Usa los botones de actualización para descargar la Base de datos con los cambios aplicados. Reemplaza el fichero original con el descargado.

          
NOTA: El orden correcto es siempre MASTER → MAT.
